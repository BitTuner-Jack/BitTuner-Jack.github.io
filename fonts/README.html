  2. Place Font Files:

    /themes/claudia/source/fonts/
└── MapleMono-Regular.woff2  ✅ (Deployed)

  3. Font Format Priority:

    • .woff2 - Modern browsers, smallest size, recommended
    • .woff - Compatible with older browsers

Other Custom Fonts

To use other fonts, follow these steps:

  1. Modify the @font-face declaration in /themes/claudia/source/style/base.scss
  2. Update font file path and font-family name
  3. Place the corresponding font files in this directory

CDN Method (Recommended)

If the font provides CDN service, you can use it directly in base.scss:

@import url('https://cdn.example.com/fonts/maple-mono.css');

This eliminates the need for local font file deployment.
